# 3D Text Animation

A Three.js project built with Vite that renders animated 3D text with matcap textures and lighting.
The project is configured to deploy automatically to GitHub Pages using GitHub Actions.

[Live Demo](https://srivathsav01.github.io/3DTextAnimation/)

---

## Tech Stack
- Three.js – 3D rendering
- Vite – Build tool & dev server
- GitHub Actions – CI/CD
- GitHub Pages – Hosting

---
## Setup
Run this followed commands:
``` bash
# Install dependencies
npm install

# Run the local server at localhost:8080
npm run dev
```
